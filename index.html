What is Css?

  1. Css stands for Cascading Style Sheet

  2. CSS describes how HTML element are to be displayed on screen,paper,or in other media

  3. css is not neccesaryCSS saves a lot of works.It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in css files.

CSS can be added to HTML Document in 3 ways

  • Inline - By using the style attribute inside HTML elements.

  • Internal - By using the style attribute in the head section.

  • External - By using a link element to link an External CSS file.

  • The most common way to add Css, is to keep the style in external css files. However,in this tutorial we will use inline and Internal style,Because this is easier to demostrate,and easier for you to try it yourself

When comparing css class vs ID,the difference is that CSS Class applies a style to multiple elements.ID,applies the style to one unique element.ID is also special in that you can use a special URL to link directly to an element and it's used by javascript.

In CSS, selectors are used to target a special element or range of element on a webpage.Once an element has been targeted,a style or set of styles can be applied to the element.

There is wide range of selectors available,Two of the most commonly use are Class and ID,Both are used target elements to which a style should be applied.
